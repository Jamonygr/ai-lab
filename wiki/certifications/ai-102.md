# AI-102 extension path

AI Lab v1 is beginner-focused, but it can become an AI-102 practice lab with optional additions.

## Covered lightly in v1

- Provision Azure AI services.
- Call Language, Vision, Speech, Translator, and Content Safety APIs.
- Use Search as a retrieval component.
- Extract layout with Document Intelligence.
- Discuss Foundry agent tool design and evaluation.
- Review private networking, Key Vault, and observability options.

## Future extensions

- Managed identity access to services.
- Private endpoints and network isolation.
- Document Intelligence.
- Azure AI Foundry project workflows.
- Prompt flow or agent exercises.
- CI checks for AI app configuration.

Do not build new Prompt Flow classic exercises for this lab. Prefer Foundry agents, direct Python calls, and explicit evaluation/tracing guidance.

## Recommended use

Use v1 to build confidence with the service surface area, then add AI-102 topics incrementally instead of turning the beginner lab into a large platform on day one.
