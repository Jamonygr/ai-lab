# AI-900 alignment

AI Lab can support Microsoft Azure AI Fundamentals study by giving learners hands-on exposure to core AI workloads.

| AI-900 area | Lab support |
|-------------|-------------|
| AI workloads and considerations | README, architecture overview, Content Safety scenario |
| Machine learning principles | Discussed conceptually; no ML workspace in v1 |
| Computer vision | Vision OCR exercise |
| Natural language processing | Sentiment, entities, PII, Translator exercises |
| Generative AI | Optional Azure OpenAI and Search/RAG extension |

## Suggested path

1. Deploy the default lab.
2. Run scenarios 1 through 6.
3. Capture output and screenshots.
4. Enable Search only after the basic API calls are clear.
5. Enable Azure OpenAI only if the subscription has access and quota.
